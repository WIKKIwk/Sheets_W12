# W12C Sheets - Project Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-27

### Added

- Initial release of W12C Sheets
- Multi-backend architecture (Go + Elixir + React)
- 100+ Excel/Google Sheets compatible formulas
- AI Assistant powered by Google Gemini 2.5 Flash
- Real-time collaboration via CRDT
- WebSocket support for live updates
- Docker and Docker Compose deployment
- Comprehensive API documentation
- Apache License 2.0

### Security

- JWT authentication
- Password hashing with bcrypt
- Rate limiting
- CORS protection
