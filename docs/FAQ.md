# Frequently Asked Questions

## General

### What is W12C Sheets?

W12C Sheets is an open-source, AI-powered spreadsheet platform similar to Google Sheets, but with self-hosting capabilities and Gemini AI integration.

### Is it free?

Yes, W12C Sheets is completely free and open-source under Apache 2.0 license.

### Can I use it commercially?

Yes, the Apache 2.0 license allows commercial use.

## Technical

### What databases are supported?

Currently PostgreSQL 15+. MySQL/MariaDB support is planned.

### Can I self-host it?

Yes! W12C Sheets is designed for self-hosting using Docker.

### What about data privacy?

All data stays on your server. We don't collect or store any user data.

## AI Features

### Do I need a Gemini API key?

Yes, each user provides their own Gemini API key for AI features.

### Are AI features required?

No, you can use W12C Sheets without AI features. It works as a regular spreadsheet.

### What AI features are available?

- Formula generation from natural language
- Data analysis and insights
- Chart recommendations
- Data cleaning suggestions

## Deployment

### What are the system requirements?

- 2 vCPU, 4GB RAM minimum
- Docker & Docker Compose
- 10GB disk space

### How long does deployment take?

About 5-10 minutes for initial setup.
