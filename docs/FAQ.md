# Frequently Asked Questions

## General

**Q: What is W12C Sheets?**
A: AI-powered spreadsheet platform similar to Google Sheets

**Q: Is it free?**
A: Yes, open-source under Apache 2.0

**Q: Can I self-host?**
A: Yes, using Docker

## Technical

**Q: System requirements?**
A: 2 vCPU, 4GB RAM, 10GB disk

**Q: Supported browsers?**
A: Chrome, Firefox, Safari, Edge (latest versions)

**Q: Can I import Excel files?**
A: Yes, via CSV conversion

## Troubleshooting

**Q: Services won't start?**
A: Check Docker is running, check logs

**Q: Database connection failed?**
A: Verify credentials in .env file
