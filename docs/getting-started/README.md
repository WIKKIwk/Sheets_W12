# Getting Started with W12C Sheets

Welcome to W12C Sheets! This guide will help you get up and running quickly.

## Quick Start

### Prerequisites
- Docker 20.10+
- Docker Compose v2+
- 4GB RAM minimum

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/WIKKIwk/Sheets_W12.git
cd Sheets_W12
```

2. **Configure environment**
```bash
cp .env.example .env
# Edit .env with your settings
```

3. **Start services**
```bash
docker compose up -d
```

4. **Access the application**
Open http://localhost:8001 in your browser
