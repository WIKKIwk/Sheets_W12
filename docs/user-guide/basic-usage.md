# Basic Usage

## Creating a Spreadsheet

1. Click "New File" button
2. Enter file name
3. Start editing cells

## Editing Cells

- Click any cell to edit
- Type value or formula
- Press Enter to save

## Formulas

Start with `=` to create a formula:
```
=SUM(A1:A10)
=AVERAGE(B1:B5)
=IF(C1>100, "High", "Low")
```

## Sharing

1. Click "Share" button
2. Enter email address
3. Set permissions (View/Edit)
4. Send invitation
