# Formula Reference

## Mathematical Functions

### SUM
Adds numbers
```
=SUM(A1:A10)
=SUM(100, 200, 300)
```

### AVERAGE
Calculates average
```
=AVERAGE(B1:B20)
```

### COUNT
Counts numeric values
```
=COUNT(C1:C50)
```

## Logical Functions

### IF
Conditional logic
```
=IF(A1>100, "High", "Low")
```

### AND
All conditions true
```
=AND(A1>0, A1<100)
```

### OR
Any condition true
```
=OR(B1="Yes", B1="Y")
```
