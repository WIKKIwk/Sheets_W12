# Keyboard Shortcuts

## Navigation
- Arrow keys: Move between cells
- Tab: Next cell
- Shift+Tab: Previous cell
- Ctrl+Home: Go to A1

## Editing
- Enter: Edit cell
- Escape: Cancel edit
- Ctrl+Z: Undo
- Ctrl+Y: Redo

## Selection
- Shift+Arrow: Extend selection
- Ctrl+A: Select all
- Ctrl+Space: Select column
- Shift+Space: Select row

## Clipboard
- Ctrl+C: Copy
- Ctrl+X: Cut
- Ctrl+V: Paste

## Functions
- Ctrl+S: Save
- Ctrl+F: Find
- Ctrl+H: Replace
