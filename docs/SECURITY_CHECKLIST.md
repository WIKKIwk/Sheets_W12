# Security Checklist

## Production
- [ ] SSL/TLS enabled
- [ ] Strong passwords set
- [ ] JWT secrets rotated
- [ ] CORS configured
- [ ] Rate limiting enabled
- [ ] Database backups automated
- [ ] Firewall configured
- [ ] Security headers enabled

## Development
- [ ] No secrets in code
- [ ] .env files in .gitignore
- [ ] Dependencies updated
- [ ] Security scanning enabled
